# Analizador-de-im-genes-termocr-micas-python-
Este script  automatiza el análisis de reflectancia de muestras termocrómicas  1. Carga una secuencia de imágenes. 2. Alinea las imágenes automáticamente usando una referencia fija  para corregir vibraciones. 3. Extrae la intensidad media  4. Convierte el tiempo a temperatura usando ecuaciones de calibración. 5. Grafica las curvas de histéresis.
